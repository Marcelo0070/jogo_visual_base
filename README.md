# Não era minha mãe – Protótipo Técnico

Protótipo técnico de jogo visual desenvolvido em [Ren'Py 8.3.7](https://www.renpy.org/), com foco em movimentação parallax dinâmica e estrutura inicial.  
Este projeto **não contém história, arte final ou elementos narrativos**, sendo uma base técnica segura para testes e desenvolvimento.

---

## 🎮 Requisitos para Executar

Para rodar este protótipo, você precisa:

- Baixar e instalar o [Ren'Py SDK 8.3.7](https://www.renpy.org/latest.html)
- Abrir a pasta do projeto pelo launcher do Ren'Py
- Clicar em **"Launch Project"** no menu

> ⚠️ Este projeto é um protótipo técnico — não é jogável como demo e não contém história ou narrativa.

---

## 🧰 Funcionalidades Técnicas

- Implementação de **parallax dinâmico** com movimento baseado no mouse
- Estrutura de labels, transições e fundo animado
- Organização modular para futuras expansões (roteiro, sprites, etc.)
